지방자치단체와 교육청의 조례, 규칙, 훈령, 예규, 고시, 의회규칙을 지역 계층과 자치법규 종류별 Markdown 파일로 제공합니다.

개요

자치법규 데이터는 지방자치단체와 교육청이 운영하는 조례·규칙 등 지역 규범을 Git으로 관리하기 위한 데이터셋입니다. 중앙 법령만으로는 확인이 어려운 지역별 운영 기준과 위임 규정을 함께 추적할 수 있습니다.

공포일자를 기준 날짜로 삼고, 광역 단위 아래에 기초자치단체, _본청, _교육청을 구분합니다. 같은 이름의 자치법규라도 지역과 종류가 다르면 별도 파일로 유지됩니다.

데이터 구조

경로는 지역 계층을 먼저 드러내도록 설계했습니다. 지역이나 교육청 단위 검색, 특정 자치법규 종류별 검색이 모두 단순한 파일 경로 필터로 가능합니다.

1단계: 광역 단위

서울특별시, 강원특별자치도처럼 시·도, 특별자치시·도, 광역연합 단위를 최상위 디렉토리로 둡니다.

2단계: 발령 주체

광역 본청은 _본청, 교육청은 _교육청, 기초자치단체는 강릉시처럼 실제 지자체명으로 구분합니다.

3단계: 자치법규 종류

조례, 규칙, 의회규칙, 훈령, 예규, 고시를 별도 디렉토리로 분리합니다.

4단계: 문서 본문

자치법규명 디렉토리 안의 본문.md가 실제 Markdown 본문입니다. 경로 전체가 안정적인 식별자 역할을 합니다.

광역 본청 조례
서울특별시/_본청/조례/{명}/본문.md
기초자치단체 규칙
경기도/수원시/규칙/{명}/본문.md
교육청 규칙
강원특별자치도/_교육청/규칙/{명}/본문.md
의회규칙
강원특별자치도/_본청/의회규칙/{명}/본문.md
교육청 훈령
강원특별자치도/_교육청/훈령/{명}/본문.md
기초자치단체 조례
강원특별자치도/강릉시/조례/{명}/본문.md

# 서울특별시 테스트 조례

Git 활용 방법

자치법규 저장소는 지역 계층과 자치법규 종류가 경로에 들어가므로 grep 명령어와 git log 명령어로 지역별 규정과 공포 이력을 좁혀 볼 수 있습니다.

자치법규 Git 활용 
grep -rl "공공시설" 서울특별시/_본청/조례/
git log -- 경기도/수원시/규칙/
git log -- 강원특별자치도/_교육청/규칙/
git log --grep="^자치법규:"

커밋 메시지

자치법규 커밋은 공포일자를 Git author/committer date로 사용하며, 자치법규명, 자치법규종류, 지방자치단체, 제개정구분, 원문 URL을 포함합니다.

예시: 자치법규 커밋 
자치법규: 서울특별시 테스트 조례 (일부개정)

자치법규 전문: https://www.law.go.kr/자치법규/서울특별시테스트조례

공포일자: 2026-01-01
자치법규종류: 조례
지방자치단체: 서울특별시

CLI 사용법

광역, 기초, 종류 필터로 목록을 좁히고 이름이나 경로로 전문을 조회할 수 있습니다. 상세한 설치와 옵션은 cli-tools 저장소를 참고하세요.

자치법규 CLI 
pip install legalize-cli
legalize ordinances list --jurisdiction 서울특별시 --type 조례 --json
legalize ordinances list --jurisdiction 강원특별자치도 --subdivision _교육청 --type 규칙 --json
legalize ordinances get "서울특별시 테스트 조례" --type 조례 --jurisdiction 서울특별시 --json
legalize search "공공시설" --in ordinances --json

MCP 사용법

MCP에서는 ordinances_list와 ordinances_get 도구를 제공합니다. 지역별 조례 검색, 교육청 규칙 조회, 특정 자치법규 전문 확인에 사용할 수 있습니다. MCP 서버 설치와 클라이언트 설정은 cli-tools 저장소를 참고하세요.
