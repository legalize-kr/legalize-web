Legalize-KR 개인정보처리방침

Legalize-KR 프로젝트는 대한민국 법령, 판례, 자치법규, 행정규칙 데이터를 생산하고 공개하는 오픈소스 프로젝트입니다.

시행일: 2026-07-01

핵심 원칙

Legalize-KR 프로젝트 자체는 법령 데이터를 생산하고 배포하기 위한 프로젝트이며, 이 과정에서 사용자의 개인정보를 수집, 보관, 이용하지 않습니다. Legalize-KR 데이터셋은 국가법령정보센터 등 공공 출처에서 제공되는 법령·판례·자치법규·행정규칙 정보를 Markdown과 Git 이력으로 정리한 것입니다.

Legalize-KR은 광고, 맞춤형 광고, 리타게팅, 사용자 프로파일링, 데이터 판매를 목적으로 사용자 개인정보를 수집하거나 이전하지 않습니다. Legalize-KR은 데이터셋을 이용하는 개인 사용자를 식별하기 위한 자체 계정, 자체 사용자 추적 ID, 자체 서버 로그 수집 체계를 운영하지 않습니다.

적용 범위

이 개인정보처리방침은 Legalize-KR 프로젝트가 공개하는 데이터셋, 오픈소스 저장소, 문서, 그리고 Legalize-KR 데이터를 활용하기 위한 도구에 적용됩니다. 예시 도구에는 cli-tools, agent-skills, viewer가 포함됩니다.

도구에서 사용자가 입력할 수 있는 정보

Legalize-KR 데이터셋 자체는 사용자 입력을 요구하지 않습니다. 다만 Legalize-KR 데이터를 편하게 조회하기 위한 일부 도구는 사용자의 실행 환경에서 다음과 같은 정보를 입력받거나 일시적으로 처리할 수 있습니다.

  • viewer: 선택적으로 입력한 GitHub 접근 토큰, 테마, 글자 크기, 즐겨찾기, 최근 문서, 로컬 브리지 URL
  • cli-tools: 사용자가 명령행에 입력한 검색어, 문서명, 사건번호, 날짜, 출력 옵션
  • agent-skills: 사용자가 LLM 또는 Agent 도구에 입력한 질문, 지시, 참고 문맥

Legalize-KR은 이러한 도구로부터 사용자 입력값, 토큰, 검색어, 프롬프트, 대화 내용, 로컬 파일 내용을 수집, 보관, 이용하지 않습니다. 도구가 저장하는 설정은 사용자의 브라우저, 로컬 파일 시스템, 명령행 환경 등 사용자가 실행하는 환경 안에 남을 수 있습니다.

제3자 플랫폼과 서비스

Legalize-KR 도구를 사용하는 과정에서 사용자가 입력한 정보가 해당 도구를 실행하는 플랫폼 또는 연동 서비스로 전송될 수 있습니다. 예를 들어 agent-skills를 LLM 서비스에서 사용하면 사용자의 질문, 지시, 참고 문맥이 해당 LLM 사업자 또는 Agent 플랫폼에 전달될 수 있습니다. Chrome Extension을 사용하면 브라우저, Chrome Web Store, GitHub, 국가법령정보센터, 사용자가 설정한 로컬 브리지 서버와 통신할 수 있습니다.

이러한 제3자 플랫폼의 데이터 처리, 보관, 학습 사용 여부, 로그 정책, 계정 정책은 Legalize-KR 프로젝트와 전혀 별개입니다. Legalize-KR은 해당 플랫폼의 개인정보 처리방침, 이용약관, 보안 설정을 통제하지 않습니다. 민감한 개인정보, 비공개 사건 정보, 접근 토큰, 업무상 비밀은 LLM 프롬프트, 공개 이슈, 명령행 로그 등에 입력하지 않는 것이 안전합니다.

GitHub와 공개 기여 기록

사용자가 Legalize-KR GitHub 저장소에 Issue, Pull Request, Discussion, 댓글, 커밋을 작성하면 그 내용과 GitHub 프로필 정보가 GitHub의 공개 기록으로 남을 수 있습니다. 이는 사용자가 GitHub에 직접 제출한 정보이며, GitHub의 개인정보 처리방침과 계정 설정이 적용됩니다.

보관과 삭제

Legalize-KR은 자체 서버에 사용자 개인정보 저장소를 운영하지 않으므로, Legalize-KR이 직접 삭제할 수 있는 사용자 개인정보 기록도 보유하지 않습니다. 각 도구가 사용자의 로컬 환경에 저장한 설정, 캐시, 토큰은 해당 도구의 설정 화면, 브라우저 저장소, 로컬 파일 삭제, 확장 제거, 플랫폼 계정 설정을 통해 삭제해야 합니다.

보안

Legalize-KR은 오픈소스 프로젝트로서 소스 코드와 데이터 생성 과정을 공개해 검토할 수 있게 합니다. 그러나 MIT License 및 Apache License 2.0을 따르는 일반적인 오픈소스 프로젝트와 마찬가지로, 소프트웨어와 데이터는 명시적 보증 없이 제공됩니다. 사용자는 자신의 실행 환경, 토큰, 계정 권한, 제3자 플랫폼 설정을 직접 관리해야 합니다.

아동의 개인정보

Legalize-KR은 아동을 대상으로 개인정보를 수집하는 서비스를 운영하지 않습니다. Legalize-KR 데이터셋과 도구는 공개 법령 정보의 조회와 분석을 위한 것입니다.

제한적 사용 준수

Chrome Extension 등 Legalize-KR 도구는 Chrome Web Store User Data Policy의 Limited Use 요구사항을 준수하도록 설계됩니다. Legalize-KR은 사용자 데이터를 광고, 데이터 판매, 신용 평가, 보험, 고용, 또는 도구의 목적과 무관한 용도로 사용하지 않습니다.
