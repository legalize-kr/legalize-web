# AI Legal Use Cases for Legalize KR

Legalize KR은 메타데이터와 날짜 이력이 필요한 AI 법률 검색, 리걸테크 서비스, 법률 데이터 분석에 활용할 수 있습니다.

## 활용 사례

1. AI 법률 검색: 법령, 판례, 행정규칙, 자치법규 원문을 근거 문서로 조회합니다.
2. RAG: Markdown 본문과 YAML frontmatter를 검색 엔진, 벡터DB, 지식 그래프에 색인합니다.
3. 날짜 기준 조회: Git 이력을 사용해 특정 날짜의 법령 또는 공공 법률문서 상태를 확인합니다.
4. MCP 도구: Claude Desktop, Cursor, VS Code, Codex 등 에이전트 클라이언트에 구조화된 조회 기능을 제공합니다.
5. 법률 데이터 분석: 법령, 판례, 기관, 지방자치단체 단위의 변경을 비교합니다.

## 중요한 경계

Legalize KR은 법률 원천 데이터를 제공합니다. 이를 기반으로 만든 시스템은 원문 조회와 법률 조언을 구분해야 하며, 답변을 생성할 때 공식 출처 URL을 함께 표시해야 합니다.
