# Legalize KR

Legalize KR은 대한민국 공공 법률문서를 Git 데이터셋으로 제공합니다. 법률 텍스트를 직접 확인해야 하는 사람과, 검색·분석·RAG에 사용할 원문 문서가 필요한 시스템을 위한 프로젝트입니다.

## 제공 범위

- YAML frontmatter가 포함된 Markdown 법령 문서.
- 사건종류와 법원등급 기준으로 분류된 판례 문서.
- 발령기관과 행정규칙 종류 기준으로 분류된 행정규칙 문서.
- 지방자치단체와 자치법규 종류 기준으로 분류된 자치법규 문서.
- 공포, 선고, 발령, 개정 사건을 Git 이력으로 보존합니다.

## 제공하지 않는 것

Legalize KR은 법률 조언 서비스가 아닙니다. 사건별 법률 의견, 변호사 매칭, 소송 대리를 제공하지 않습니다. 검색, 인용, 연구, 자동화에 활용할 수 있는 공공 법률 데이터셋입니다.
