모든 개정, 추적됩니다.

실제 법령 개정 내용을 diff로 확인합니다. PDF를 비교하거나 요약본에 의존할 필요가 없습니다.

민법 개정 사항 비교 민법 제809조
-##### 제809조 (동성혼등의 금지)
+##### 제809조 (근친혼 등의 금지)
-**①** 동성동본인 혈족 사이에서는 혼인하지 못한다.
+**①** 8촌 이내의 혈족(친양자의 입양 전의 혈족을 포함한다) 사이에서는 혼인하지 못한다.
-**②** 남계혈족의 배우자, 부의 혈족 및 기타 8촌 이내의 인척이거나 이러한 인척이었던 자 사이에서는 혼인하지 못한다.
+**②** 6촌 이내의 혈족의 배우자, 배우자의 6촌 이내의 혈족, 배우자의 4촌 이내의 혈족의 배우자인 인척이거나 이러한 인척이었던 자 사이에서는 혼인하지 못한다.
2005-03-31 민법 개정 — 동성동본 금혼 폐지, 근친혼 금지로 전면 개정

// 어떻게 작동하나

국가법령정보에서 Git으로.

공식 데이터 출처

국가법령정보센터 OpenAPI에서 법령과 판례를 가져옵니다. 항상 공식 원문을 기반으로 합니다.

구조화 + 버전 관리

각 법령과 판례는 YAML 메타데이터가 포함된 Markdown 파일이 됩니다. 각 개정과 판결은 실제 날짜를 가진 Git 커밋입니다.

활용하세요

grep으로 법령과 판례를 검색하고, git log로 이력을 확인하고, git diff로 개정 내용을 비교하세요.

// 법령 파일 구조

법령명으로 정리되어 있습니다.

관련 법령(법률, 시행령, 시행규칙)이 하나의 디렉토리에 함께 관리됩니다.

법률
kr/{법률명}/법률.md
시행령
kr/{법률명}/시행령.md
시행규칙
kr/{법률명}/시행규칙.md
독립 대통령령
kr/{대통령령명}/대통령령.md

// 판례 파일 구조

사건 종류별로 정리되어 있습니다.

판례는 사건종류와 법원등급(대법원, 하급심)으로 분류되어, 사건번호 단위의 Markdown 파일로 관리됩니다.

민사

형사
형사/{법원등급}/{사건번호}.md
일반행정
일반행정/{법원등급}/{사건번호}.md
그 외
