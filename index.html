// 무엇이 바뀌었나

모든 개정, 추적됩니다.

실제 법령 개정 내용을 diff 형식으로 확인합니다. PDF를 비교하거나 요약본에 의존할 필요가 없습니다.

민법 개정 사항 비교 민법 제809조
-##### 제809조 (동성혼등의 금지)
+##### 제809조 (근친혼 등의 금지)
-**①** 동성동본인 혈족 사이에서는 혼인하지 못한다.
+**①** 8촌 이내의 혈족(친양자의 입양 전의 혈족을 포함한다) 사이에서는 혼인하지 못한다.
-**②** 남계혈족의 배우자, 부의 혈족 및 기타 8촌 이내의 인척이거나 이러한 인척이었던 자 사이에서는 혼인하지 못한다.
+**②** 6촌 이내의 혈족의 배우자, 배우자의 6촌 이내의 혈족, 배우자의 4촌 이내의 혈족의 배우자인 인척이거나 이러한 인척이었던 자 사이에서는 혼인하지 못한다.
2005-03-31 민법 개정 — 동성동본 금혼 폐지, 근친혼 금지로 전면 개정

// 어떻게 작동하나

국가법령정보에서 Git으로.

공식 데이터 출처

국가법령정보센터 OpenAPI에서 법령, 판례, 자치법규, 행정규칙을 가져옵니다. 항상 공식 원문을 기반으로 합니다.

구조화 + 버전 관리

각 문서는 YAML 메타데이터가 포함된 Markdown 파일이 됩니다. 개정, 판결, 공포, 발령 이력은 실제 날짜를 가진 Git 커밋입니다.

활용하세요

grep 명령어로 4종류의 데이터셋을 검색하고, git log 명령어로 이력을 확인하고, git diff 명령어로 변경 내용을 비교하세요.

// GitHub API 기반 조회

CLI로 바로 조회합니다.

저장소를 통째로 복제하지 않아도 법령, 판례, 자치법규, 행정규칙을 같은 명령 체계로 조회할 수 있습니다. 자동화에는 --json 출력을 사용합니다. 상세한 설치와 옵션은 cli-tools 저장소를 참고하세요.

legalize-cli
$ pip install legalize-cli
$ legalize laws get 민법 --date 2015-06-01
$ legalize precedents list --court 대법원 --type 민사 --json
$ legalize ordinances list --jurisdiction 서울특별시 --type 조례 --json
$ legalize admrules list --agency 행정안전부 --type 고시 --json
$ legalize search "개인정보" --in all --json
