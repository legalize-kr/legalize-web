Legalize-KR Viewer

Legalize-KR Viewer는 법령, 판례, 자치법규, 행정규칙 Markdown Git 저장소를 브라우저에서 읽고 변경 이력을 비교하는 읽기 전용 Chrome Extension입니다.

개인정보처리방침 Legalize KR 활용법

주요 기능

문서 탐색

법령, 판례, 자치법규, 행정규칙 저장소의 Markdown 문서를 제목과 경로 기준으로 탐색합니다.

변경 내역

Git 커밋 이력을 불러와 개정·선고·공포·발령 흐름을 확인하고 문서 차이를 비교합니다.

첨부 보기

국가법령정보센터에서 제공하는 PDF, HWP, HWPX 첨부를 브라우저 안에서 열람합니다.

로컬 소스

필요하면 로컬 폴더 또는 로컬 브리지 서버를 통해 복제한 저장소를 읽을 수 있습니다.

데이터 처리 원칙

확장은 Legalize-KR 원천 저장소를 수정하거나 원격에 업로드하지 않습니다. GitHub 접근 토큰은 선택 사항이며, 사용자가 저장한 경우 GitHub API 요청 한도 완화에만 사용합니다.

자세한 저장 항목, 전송 대상, 삭제 방법은 Legalize-KR 개인정보처리방침에서 확인할 수 있습니다.
